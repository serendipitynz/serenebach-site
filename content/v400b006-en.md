Serene Bach v4.0.0-beta.6 has been released.

* New Feature: Added support for user-defined `{custom_xxx}` placeholders in sbtemplate.
* New Feature: Archive boundaries and entry dates are now anchored to a configurable timezone.
* New Feature: Added HTTP server timeouts and graceful shutdown.
* Security: Switched static file serving for `/img` and `/template` to `os.Root`, hardening it against directory traversal.
* Performance: Cached sbtemplate parse results to avoid re-parsing on every page render.
* Internal: Reorganized sbtemplate tag rendering so that HTML escaping is the default.

Download: [Serene Bach v4.0.0-beta.6](https://github.com/serendipitynz/serenebach/releases/tag/v4.0.0-beta.6)

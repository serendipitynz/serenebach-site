Serene Bach v4.0.0-beta.2 has been released.

* New Feature: Added MCP OAuth Proxy for ChatGPT integration.
* New Feature: Added Docker support.
* Development: Enabled Hot Reload for the admin panel when running `task dev`.
* Bug Fix: Fixed an issue where the item order in certain admin screens was incorrect.
* New Feature: Enabled Drag & Drop reordering within Link Groups.
* Change: AI-generated suggestions are now displayed in a popup instead of being inserted directly into the editor.

Download: [Serene Bach v4.0.0-beta.2](https://github.com/serendipitynz/serenebach/releases/tag/v4.0.0-beta.2)

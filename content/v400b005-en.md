Serene Bach v4.0.0-beta.5 has been released.

* New Feature: Added flat pages, with OG card support, admin CRUD, per-author ownership, and integration with the static rebuild pipeline.
* New Feature: Added a `toppage` template block for content that should only appear on the top page.
* New Feature: Profile pages now also use the template selected in the design settings.

Download: [Serene Bach v4.0.0-beta.5](https://github.com/serendipitynz/serenebach/releases/tag/v4.0.0-beta.5)

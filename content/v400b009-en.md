Serene Bach v4.0.0-beta.9 has been released.

* New Feature: Category URLs and snapshot paths now use the category slug.
* New Feature: Categories can now be marked as hidden, useful for keeping archives without showing them in the main timeline.
* Bug Fix: Honor SB3's per-category entry template priority.
* Bug Fix: `{entry_date}` on list pages now follows the entry date format (DateFormatEntry).
* Bug Fix: Inline dates in the sidebar (latest entries and latest comments) now follow the list date format (DateFormatList), matching SB3 behavior.

Download: [Serene Bach v4.0.0-beta.9](https://github.com/serendipitynz/serenebach/releases/tag/v4.0.0-beta.9)

Serene Bach v4.0.0-beta.19 をリリースしています。

- 機能追加: デフォルトテンプレートに検索 UI を追加しました
- テンプレート: go.serenebach.net で配布しているテンプレートにも同じ変更を適用し、default 系および monochrome テンプレートに検索フォームとエントリ別 SEO メタブロックを追加しました
- その他: GitHub Actions のランタイムを Node.js 24 に更新しました

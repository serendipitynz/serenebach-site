Information on 2026-05-04

Serene Bach v4 is a self-hosted blog engine rebuilt with Go and SQLite, while preserving the familiar user experience of Serene Bach v3.

Some features previously supported in Serene Bach v2 and v3, such as trackbacks and XML-RPC, have been intentionally removed. Other features, such as search, have not been implemented yet.

On the other hand, Serene Bach v4 also includes several more modern features:

  • Markdown support
  • Generation of Open Graph images for individual articles
  • Responsive design and dark-mode support in the default template
  • AI integration for tasks such as generating titles or summarizing selected text
  • Docker-based deployment

Although Serene Bach v4 has been rebuilt in Go, it still supports running as a CGI program.
